Zusammenfassung

This is an example ontology to help PMD partner projects to get an overview over the formal requirements imposed by HAP 4. It's meant as a blueprint for updating the partner projects ontologies and thus does not carry any relevant concepts. This file is related to the github repository https://github.com/materialdigital/ontology_publication_template where there is more information about how to work with this example.
